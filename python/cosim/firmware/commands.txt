List of control transfer types

Program flow:
    Software configuration
    -> Converter register values
    -> I/O module register values
    -> HWCON values
    

Control transfers may be sent to or from the FPGA over USB.  Direction choices are:
    - I: In, from the FPGA to the computer
    - O: Out, from the computer to the FPGA
    - I/O: Either direction

Control transfers are each designated by a 1-byte code:
    - The first bit determines whether action is requested:
      Commands have codes starting with 0 (0x00 - 0x7F)
      Status/informational packets start with 1 (0x80 - 0xFF)

Packet format:
    Byte 0: header byte (0xFF default)
    Byte 1: transfer type (0x00 - 0xFF)
    Bytes [3:2]: length N in bytes
    Bytes 4 to N+3: data

0x00: No operation (data length 0, ignore)

Meta commands (0x0x)
Hex code    Desc                            Dir Data length     Data format
0x01        Select source                   I   1               ID of source
0x02        Get current source              I   0
0x03        Go to next source               I   0

Seek commands (0x1x)
Hex code    Desc                            Dir Data length     Data format
0x10        Seek to track                   I   1               Track
0x11        Seek prev track                 I   0
0x12        Seek next track                 I   0
0x13        Seek to disk/playlist/album     I   1               ID of disk/playlist/album
0x14        Seek prev playlist              I   0
0x15        Seek next playlist              I   0
0x16        Fast forward step               I   0
0x17        Fast rewind step                I   0
0x18        Seek to time                    I   5               Port, 32-bit time code
            

Power management commands (0x2x)
Hex code    Desc                            Dir Data length     Data format                     Notes
0x20        Standby                         I/O 0                                               Implement later
0x21        Shutdown                        I/O 0                                               Implement later
0x22        Wakeup                          I/O 0                                               Implement later

Configuration commands (0x3x) - Note: only pertains to configuration needed for audio conversion.
                                These values may be altered as consequences of driver settings (i.e. oversampling).
Hex code    Desc                            Dir Data length     Data format
0x31        Configuration: get value        O   2               Port, address
0x32        Configuration: set value        O   4               Port, address, 16-bit value
0x33        Configuration: get everything   O   0               N/A
0x34        Set HWCON register              O   2               Port, 8-bit value
0x35        Read HWCON register             O   1               Port
0x36        Set I/O module register         O   4               Port, address, 16-bit value
0x37        Read I/O module register        O   1               Port
0x38        Enable SPI controller           O   0               N/A
0x39        Disable SPI controller          O   0               N/A
            
Synchronization commands (0x4x)
Hex code    Desc                            Dir Data length     Data format
0x40        Set timebase                    O   5               Port, 32-bit time code
0x41        Flush buffers                   O   1               Port
            
Status/informational packets (0x8x)
Hex code    Desc                            Dir Data length     Data format
0x80        Current time codes              I   5N              Port, 32-bit time code (may be repeated)            
0x81        Current audio source            I   1               ID of source
0x82        Current hardware register       I   4               Port, address, 16-bit value
0x83        Current HWCON register          I   2               Port, 8-bit value
0x84        Current I/O module register     I   4               Port, address, 16-bit value
