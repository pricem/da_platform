
Target - 1 byte
    00, 01, 02, 03 = slots
    OR, FF = global

Slot-specific commands
    Write to data FIFO - 3 bytes for length, then data
    Write to command FIFO - 3 bytes for length, then data

Global commands
    Set clock sel (1 byte data, lower 4 bits control clock sel for 4 slots)
    Get overflow status
    Get dir/chan
