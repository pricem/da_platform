"Samoyed" digital audio platform
=======================

This page is the starting point for documentation about the project (often referred to as the "DA platform").

.. figure:: photos/DSC_0273.jpg
    :width: 75%
    :align: center

    A baseline implementation of the platform.  This one has two AK4490-based 2-channel DACs, as well as an 8-channel DAC and 8-channel ADC.  The onboard Raspberry Pi functions as a music streaming server and digital active crossover.
